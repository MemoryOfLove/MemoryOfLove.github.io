# Connect 函数的用法：

第二个参数是回调函数，回调函数的参数根据第一个参数的属性来设置。需要注意的是，第二个参数所设置的回调函数，其第一个参数是 string 类型的 context。

# ConnectWithoutContext 的用法：

第二个参数是回调函数，参数的设置依赖与第一个参数设置的属性。与 Connect 函数不同的是，这里的回调函数的第一个参数没有 string 类型的 context。
